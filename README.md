# Expense Tracker - MERN Stack Application

A full-featured expense tracking application built with the MERN stack (MongoDB, Express.js, React.js, Node.js) that helps users manage their finances, track expenses, set budgets, and achieve savings goals with AI-powered insights.


## Features

### Core Features
- **User Authentication** - Secure JWT-based authentication with password encryption
- **Transaction Management** - Add, edit, delete income and expense transactions
- **Balance Tracking** - Real-time calculation of total balance (Income - Expense)
- **Responsive Design** - Fully responsive UI that works on desktop, tablet, and mobile

### Intermediate Features
- **Category System** - Pre-defined categories (Food, Travel, Bills, Shopping, etc.)
- **Date Filters** - Filter transactions by date range
- **Monthly Summary** - Visual representation of monthly income vs expenses
- **Charts & Graphs** - Interactive pie charts and bar charts using Recharts

### Advanced Features
- **Analytics Dashboard** - Income vs expense insights and spending patterns
- **Search Transactions** - Search through transaction history
- **Export Data** - Export transactions to CSV and PDF formats
- **Dark Mode** - Toggle between light and dark themes
- **Mobile Responsive** - Optimized for all screen sizes

### Unique Features (AI-Powered)
- **AI Suggestions** - Smart insights about spending patterns
  - "You spent 30% more on food this month"
  - "You've used 90% of your entertainment budget"
- **Budget Alerts** - Get notifications when nearing budget limits
- **Savings Goals** - Set and track progress towards financial goals

## Technology Stack

### Frontend
- **React.js** - UI library
- **Vite** - Build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Material-UI** - React component library
- **Recharts** - Charting library
- **Axios** - HTTP client
- **React Router DOM** - Routing
- **React Hot Toast** - Toast notifications
- **React Icons** - Icon library

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - ODM for MongoDB
- **JWT** - Authentication
- **bcryptjs** - Password hashing
